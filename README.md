# Personal Portfolio 🚀

A clean, responsive, and professional portfolio website built to showcase my projects and skills as a BSc CSIT student.

**🔗 Live Site:** [khanalsafal.com.np](https://www.khanalsafal.com.np/)

## ✨ Features
- **Fully Responsive:** Optimized for all devices (Desktop to Mobile).
- **Dark Mode:** Seamless theme switching for a better user experience.
- **Glassmorphism:** Elegant blur effects on navigation and UI elements.
- **Sticky Navigation:** Smooth scrolling and easy access to sections.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6)
- **Icons:** Font Awesome
- **Deployment:** Vercel with a custom `.com.np` domain
